Write a program that takes a single string as input representing a user's comment. Your task is to check if the comment follows the Kotlin style guide for single-line comments. If the comment is valid, print "Valid", otherwise print "Invalid". A valid single-line comment in Kotlin should start with "//" followed by a space.


Sample Input:
// This is a valid comment

Sample Output:
Valid


Sample Input:
//Invalid comment

Sample Output:
Invalid



Memory limit: 256 MB
Time limit: 8 seconds
