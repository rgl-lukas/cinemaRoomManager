Write a program that reads a string from the user and checks if it is a palindrome. A palindrome is a string that reads the same forwards and backwards, ignoring case and whitespace. Print 'true' if the string is a palindrome, and 'false' otherwise.
Sample Input:
racecar

Sample Output:
true


Sample Input:
A man, a plan, a canal: Panama

Sample Output:
true
